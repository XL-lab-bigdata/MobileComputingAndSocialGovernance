
人口流动是由经济、社会、环境等多方面因素驱动的社会现象，其时空演化规律深刻塑造着城市治理的底层逻辑。从 1790 年美国完成第一次现代人口普查，到 21 世纪智能手机的信令追踪，人类对流动性的认知经历了从宏观统计到微观解构的范式跃迁。

人口受到生产要素的影响在时空中流动，人口流动反之推动了生产要素的再次集聚与扩散，从而成为推动社会经济发展的重要内在动力。本篇详细阐述移动定位大数据在人口流动研究中的潜力和重要性，包括数据来源、处理及网络构建等关键环节。在此基础上，通过时序特征、空间分布及季节性人口流动时空演化模式等分析，揭示了我国人口流动的时空演化模式。本篇还深入探讨我国人口流动的影响因素，分析人口流动时空模式，挖掘城市群结构动态变化等，展现了移动定位大数据在人口流动研究中的巨大价值，为科学规划、优化资源配置及应对重大事件提供新视角和科学方案。
