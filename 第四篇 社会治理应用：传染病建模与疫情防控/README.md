
作为人类与病原体协同演化的复杂系统，传染病传播在人类历史的文明进程和社会发展中扮演了重要角色。当移动数据突破传统流行病学调查的时空局限，传染病防控正经历从“经验驱动”到“数据智能”的认知革命。

本篇构建了“感知—预警—干预”的传染病智能防控策略。在公共卫生角度，传染病防控涉及控制传染源，切断传播路径，保护易感人群等多种措施。移动计算利用数字化追踪技术对移动轨迹溯源，构建人口流动网络对传染病进行传播预测和风险评估。在此基础上，从三个案例出发，以海地霍乱疫情，西非埃博拉疫情，全球新冠疫情为背景，描述了移动数据获取和处理方式，通过人类在紧急事件下的流动行为揭示出疫情传播模式，从而科学地制定疫情防控应对策略。移动计算技术正在重构公共卫生防线。当每个移动设备成为防疫网络的关键节点，人类首次获得在病原体进化竞赛中实现“数字免疫”的战略能力。
