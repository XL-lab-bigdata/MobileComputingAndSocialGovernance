**吕欣，李云轩 等 著。移动计算与社会治理。科学出版社，2026 (ISBN 978-7-03-083212-2)**

每天，我们都在移动。

从清晨的上班通勤，到节假日的人口迁徙；

从城市中的潮汐流动，到突发灾害后的紧急撤离；

从疫情中的风险扩散，到公共事件后的社会响应——

人的移动，构成了现代社会最真实、最复杂、也最值得理解的运行图景。

<img width="1000" height="563" alt="ddda0162bf5f2de57a34a71bb0855380" src="https://github.com/user-attachments/assets/ae221de2-a4de-4def-8b80-a0a4a8e889f5" />

过去，我们常常依靠统计报表、调查问卷和经验判断来理解社会。

而今天，移动互联网、物联网、大数据和人工智能的发展，让我们有机会以更高的精度、更广的尺度、更快的速度，重新观察社会运行的深层规律。

社会治理面对的，也不再只是固定空间中的静态问题，而是一个由人、地点、时间、网络和事件共同交织而成的复杂系统。

移动计算的价值，正在于此。它可以帮助我们理解：

**人从哪里来，又到哪里去？**

**城市之间如何形成联系？**

**疫情风险如何沿着人口流动网络传播？**

**灾害发生后，人群如何避险、撤离和重新分布？**

**公共资源应该如何更及时、更精准地配置？**

从这一意义上说，移动计算已经超越了单纯的技术工具。它更像是数字社会的“感知系统”，在严格隐私保护和匿名处理的前提下，帮助我们从海量移动数据中识别社会运行的节律、结构与异常变化。

<img width="800" height="1067" alt="f2271132c9d7c3c76fb2a719ae93da7f" src="https://github.com/user-attachments/assets/d52b0e4c-ec85-4887-9343-ff5275c7b474" />

人的移动，看似日常，却从来不是孤立的。

我们可以从中看到城市的节律：哪里在清晨被唤醒，哪里在夜晚仍然活跃；

也可以看到区域的联系：哪些城市彼此依赖，哪些空间正在形成新的流动走廊；

还可以看到风险的路径：疾病、灾害、资源短缺和突发事件，如何沿着人口流动网络扩散、放大或被阻断。

<img width="1000" height="1000" alt="fbb544f0e94a7fdb1f3e35296472bb00" src="https://github.com/user-attachments/assets/ebe2449a-230d-4830-acdd-9ae90f20c904" />

移动计算要做的，不是把社会压缩成数据，而是借助数据重新理解社会：

理解那些隐藏在人群流动背后的秩序，理解城市运转中的节奏，理解流行病风险传播前的蛛丝马迹，理解治理行动可以提前抵达的方向。

**它记录的是人的活动，折射的是城市结构，连接的是公共安全、资源配置和社会韧性。**

本书的构思与撰写，源于近年来移动互联网、物联网、大数据与人工智能技术的飞速发展给社会治理带来的深刻变革与崭新契机。国内外相关机构与团队已围绕社会系统高精度感知、大规模人类行为规律挖掘、突发事件应急响应等方向，开展了一系列代表性前沿实践，为新时代社会治理的创新突破与发展指明了重要方向。本书整体内容包括：

• **移动技术基础篇**系统梳理了从模拟信号的0G 时代到高速泛在的5G 乃至未来通信技术的演进脉络，详解各代网络架构核心，并前瞻卫星通信等前沿方向；深入阐释移动大数据的定义、独特特征与主要类型，剖析其存储管理技术体系与典型平台；重点探讨移动大数据隐私与安全的核心挑战，涵盖K 匿名、差分隐私、同态加密等关键防护算法，为安全、合规地利用移动数据价值奠定基础。

• **移动计算理论篇**深入剖析移动定位技术原理体系；系统构建个体与群体移动轨迹的模型化表达框架（涵盖随机游走、列维飞行到探索偏好返回、引力模型、辐射模型等）；运用复杂网络理论，构建移动网络的结构刻画方法、核心统计指标（如中心性、集聚系数、效率）以及社区挖掘与时序分析技术，为理解移动连接背后的社会结构与动态模式提供强有力的理论工具。

• **社会治理应用篇**聚焦三大实践场景：

人口流动管理聚焦城市化进程的核心命题。我们如何量化亿万人潮汐般的迁徙规律？如何从海量移动定位数据中，洞察城市群的空间结构、活力边界与交互强度？本篇通过构建精细的人口流动时空演化模式分析框架、影响因素模型及网络社区挖掘方法，致力于揭示区域协同发展的内在逻辑，为优化国土空间布局、提升城市治理效能提供科学依据。

传染病建模与疫情防控直面公共卫生安全的时代挑战。当疫情来袭，如何利用移动大数据精准刻画风险传播路径？如何评估不同防控策略（如出行限制、接触者追踪）的有效性？本篇以海地霍乱、西非埃博拉、全球新冠病毒等重大疫情为实战沙盘，深入探讨基于人口流动网络的传播动力学建模、风险评估模型与多智能体仿真技术，展现了移动计算在预警预测、资源调配、政策评估中不可替代的“哨兵”与“智囊”作用。

自然灾害应急救援诠释移动计算如何成为应急响应先锋。在灾难突发的危急时刻，如何快速评估灾情影响范围？如何追踪受灾人群的动态分布与撤离轨迹，实现救援力量的精准投送？本篇结合海地地震、孟加拉气旋风暴、尼泊尔震洪复合灾害等典型案例，详细解析了如何利用移动通信数据实时感知灾后人口异常流动、分析行为模式变化、评估脆弱性，为构建快速响应、科学决策、高效协同的现代化应急救援体系点亮“智慧之眼”。
