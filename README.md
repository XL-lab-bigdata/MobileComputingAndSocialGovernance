**吕欣，李云轩 等 著。移动计算与社会治理。科学出版社，2026 (ISBN 978-7-03-083212-2)**
<img width="430" height="531" alt="移动计算与社会治理-立封" src="https://github.com/user-attachments/assets/9f102a4b-d8f9-4a55-95f6-84d490295ffb" />

每天，我们都在移动。

从清晨的上班通勤，到节假日的人口迁徙；

从城市中的潮汐流动，到突发灾害后的紧急撤离；

从疫情中的风险扩散，到公共事件后的社会响应——

人的移动，构成了现代社会最真实、最复杂、也最值得理解的运行图景。

<img width="1000" height="563" alt="ddda0162bf5f2de57a34a71bb0855380" src="https://github.com/user-attachments/assets/ae221de2-a4de-4def-8b80-a0a4a8e889f5" />

过去，我们常常依靠统计报表、调查问卷和经验判断来理解社会。

而今天，移动互联网、物联网、大数据和人工智能的发展，让我们有机会以更高的精度、更广的尺度、更快的速度，重新观察社会运行的深层规律。

社会治理面对的，也不再只是固定空间中的静态问题，而是一个由人、地点、时间、网络和事件共同交织而成的复杂系统。

移动计算的价值，正在于此。它可以帮助我们理解：

**人从哪里来，又到哪里去？**

**城市之间如何形成联系？**

**疫情风险如何沿着人口流动网络传播？**

**灾害发生后，人群如何避险、撤离和重新分布？**

**公共资源应该如何更及时、更精准地配置？**

从这一意义上说，移动计算已经超越了单纯的技术工具。它更像是数字社会的“感知系统”，在严格隐私保护和匿名处理的前提下，帮助我们从海量移动数据中识别社会运行的节律、结构与异常变化。

<img width="800" height="1067" alt="f2271132c9d7c3c76fb2a719ae93da7f" src="https://github.com/user-attachments/assets/d52b0e4c-ec85-4887-9343-ff5275c7b474" />

人的移动，看似日常，却从来不是孤立的。

我们可以从中看到城市的节律：哪里在清晨被唤醒，哪里在夜晚仍然活跃；

也可以看到区域的联系：哪些城市彼此依赖，哪些空间正在形成新的流动走廊；

还可以看到风险的路径：疾病、灾害、资源短缺和突发事件，如何沿着人口流动网络扩散、放大或被阻断。

<img width="1000" height="1000" alt="fbb544f0e94a7fdb1f3e35296472bb00" src="https://github.com/user-attachments/assets/ebe2449a-230d-4830-acdd-9ae90f20c904" />

移动计算要做的，不是把社会压缩成数据，而是借助数据重新理解社会：

理解那些隐藏在人群流动背后的秩序，理解城市运转中的节奏，理解流行病风险传播前的蛛丝马迹，理解治理行动可以提前抵达的方向。

**它记录的是人的活动，折射的是城市结构，连接的是公共安全、资源配置和社会韧性。**

看见流动只是第一步。真正困难的问题在于：如何从海量、复杂、动态的数据中，提炼出可以解释、可以预测、可以服务决策的知识？

**移动定位技术，让我们能够更精细地捕捉人在空间中的位置变化。**

从蜂窝基站到卫星导航，从Wi-Fi、蓝牙到RFID，位置的获得并不是一个简单的“定位”动作。信号如何被接收，距离如何被估计，位置如何被推算，误差又如何被理解？这些技术问题共同构成了移动计算感知世界的第一步。也正因为有了这些基础，人的移动才第一次能够被连续、细致地放入空间之中。

<img width="1126" height="562" alt="cce8cdee3a47b61e7ef75052a72d11e9" src="https://github.com/user-attachments/assets/bde09673-0886-4ed1-99ad-8cada805250d" />

**移动轨迹挖掘，让看似杂乱的移动行为呈现出可分析的模式。**

一条轨迹看上去只是若干位置点的连线，但当时间、频率、停留、范围和复杂度被纳入分析，它就开始呈现出行为的纹理。那些反复出现的路径、熟悉的停留地点、周期性的出行节奏，逐渐从杂乱的数据中浮现出来。人的移动并非完全随机，日常生活中看似平常的往返、停留与转向，往往隐藏着稳定的行为规律。

<img width="1120" height="650" alt="03f016ad8e8012e812b53aa4947adb4f" src="https://github.com/user-attachments/assets/067286b3-ca91-4d04-bca5-db09948cb07f" />

**人口流动模型，让城市之间、区域之间的联系不再只是经验判断，而可以被量化、比较和推演。**

当一个人的轨迹汇入千万人的移动，个体行为便上升为人口流动。流动为什么发生，强度如何形成，距离、机会和城市吸引力如何共同影响人群选择？借助模型，城市之间的联系不再只是地图上的相邻关系，而成为可以计算、比较和推演的社会联系。

<img width="994" height="680" alt="eb9eb7f2a12e413a6c65f489ada531b0" src="https://github.com/user-attachments/assets/6384f223-1b40-41f0-9122-d4d86783ca48" />

**复杂网络理论，则进一步把个体、地点和城市之间的关系组织成一张可以被理解的社会连接图。**

人口、地点、城市之间的移动连接，被组织成一张不断变化的网络：有的节点承担枢纽功能，有的区域形成紧密社区，有的联系在特定时间突然增强。通过这样的视角，我们不仅能看到谁与谁相连，更能理解社会系统如何通过流动形成结构。

<img width="1031" height="617" alt="8d1df46a2270cf61b480a060be5c16b0" src="https://github.com/user-attachments/assets/01898119-39fc-4255-98b9-bf048f3cd48d" />

于是，移动不再只是移动。

它可以**成为城市空间结构的证据**。

在人口流动管理中，移动数据不只是记录人群迁徙，而成为理解城市体系和区域关系的证据。哪些城市正在形成更强联系，哪些区域正在出现新的流动通道，哪些空间结构正在被人口活动重新塑造，都可以在流动网络中找到线索。城市不只是被建设图纸规划出来的，也是在一次次真实移动中被重新定义的。

<img width="1161" height="1327" alt="e1d1795d36a9778b6672b85c257af277" src="https://github.com/user-attachments/assets/89c527f0-2fff-4226-a0ee-fd1fa2f27971" />

它可以**成为公共卫生风险评估的基础**。

在公共卫生场景中，流行病的传播不只发生在病原体与个体之间，更发生在人群流动、社会关系和区域连接之中。人口从哪里流出，风险就可能沿着哪里扩散；人群在哪里汇聚，防控压力就可能在哪里累积。通过流动网络、传播模型和政策仿真，我们可以更早识别风险扩散方向，也可以更科学地评估防控措施可能带来的影响，制定精准的公共卫生防控策略。

<img width="1298" height="1280" alt="2b6880b90ee2efd2a2c128ad03d4a579" src="https://github.com/user-attachments/assets/3b3a657f-7468-4944-96d6-c18138556eb1" />

它可以**成为灾害应急响应中的实时线索**。

在灾害应急响应中，移动数据能够帮助我们在混乱中重建秩序。当地震、气旋、洪水等灾害突然发生，人口流动会迅速偏离日常状态：有人撤离，有人滞留，有人返回，也有人进入新的风险区域。通过这些异常变化，治理者可以更及时地判断受灾影响范围、识别撤离路径、发现脆弱人群，并为救援资源投送争取宝贵时间。

<img width="841" height="1035" alt="4b481b145b3a8f90c4f8f88c647a4608" src="https://github.com/user-attachments/assets/74f35866-07f8-41db-8754-785556ae463b" />

《移动计算与社会治理》所呈现的，正是这样一条从技术走向治理、从数据走向行动的路径。它不是一本停留在概念介绍中的书，而是把移动通信、轨迹建模、复杂网络、人口流动、疫情防控、灾害应急等内容连接起来，带领读者重新理解数字时代社会运行的底层逻辑。

对于管理科学、信息科学和社会学研究者，它提供了一个跨学科的知识框架；

对于社会治理、应急管理、公共卫生实践者，它提供了一套理解复杂社会问题的计算视角；

对于关心智慧城市、公共安全与数字社会未来的读者，它提供了一种新的观察方式。

每一次移动，都是社会运行留下的线索；

每一种流动，都可能成为理解治理的入口。
